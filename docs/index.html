Your adventure starts here. Before you head out, you'll need to install one small app on your phone.

It takes about two minutes. Follow the steps below.

1. Download the app

Tap the button below to download the file. It will land in your Downloads folder.

Download

2. Allow installation

When you open the file, Android will ask if you want to allow installation from this source. This is expected — tap Allow.

If it asks you to enable unknown sources first:

  1. Tap Settings
  2. Toggle on Allow from this source
  3. Head back and tap Install

3. Open the app and go

Once installed, open the app and allow it to access your location when prompted.

Then tap Let's Go.
